A clean, simplified WordPress Admin theme.

== Description ==

Webseo provides a clean, simplified design for your WordPress Admin area. 

Our goal was to simplify the visual design with a primary focus on the content writing experience.

If you like Webseo, please consider leaving a review here on WordPress. Thank you!

== Installation ==

1. Unzip `Webseo-admin-theme.zip`
2. Upload the `Webseo-admin-theme' folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
